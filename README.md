# Food-Delivery-App


## Installation

Start by cloning the repo to your personal machine. Make sure you have react native and Node installed.

### Run Expo

`npx expo start`

### Run Sanity

```
cd delivery-app
npm run dev
```

## Technologies Used

- React Native 
- Expo
- Redux
- NativeWind / TailwindCSS
- Sanity.io - building schemas
