# Food-Delivery-App


## Installation

Start by cloning the repo to your personal machine. Make sure you have react native and Node installed.

### Run app through Expo

`npx expo start`

### Run sanity

```
cd delivery-app
npm run dev
```

### Deploy

`npm run deploy`

## Technologies Used

- React Native 
- Expo
- Redux
- TailwindCSS
- Sanity.io - building schemas
